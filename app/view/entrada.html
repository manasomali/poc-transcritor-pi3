A plataforma web para Registro de Transcrições de Voz de Operadores do Setor Elétrico é uma solução voltada para operadores e colaboradores da pós operação. A plataforma proposta é inovadora pois atualmente não existe no mercado um sistema de transcrição e registro especificamente para operadores do SIN.

Os principais frameworks ultilizados na ferramenta computacional são:
• AngularJS Framework;
• Bootstrap Framework;
• JQuery Framework;
• Handsontable;
• MySQL Database;
• P5.Speech;
• Algoritmos desenvolvidos em JavaScript;
• Algoritmos desenvolvidos em PHP.
